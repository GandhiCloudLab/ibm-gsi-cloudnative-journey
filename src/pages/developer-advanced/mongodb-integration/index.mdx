<PageDescription>

Add the MongoDB using operator

</PageDescription>

- Open the Openshift Web Console.

- Choose the Developer option.

- Click on the Add on the left-hand menu and Choose Database.

 ![Database](/images/database-catalog.png)

- Select  the MongoDB Tile on the Developer Catalog.

 ![MongoDB](/images/mongodb.png)

- Clicking on MongoDB its open the template page. Click the Instantiate Template button.

- Open MongoDB Configuration page. Enter the necessary infomation such as `Database Service Name, MongoDB Connection Username, MongoDB Connection Password, MongoDB Database Name, MongoDB Admin Password, Volume Capacity etc..`.

- Click create button and it will create MongoDB Instance.

 ![Db Configuration](/images/mongodb-configuration.png)

- This MongoDB Operator create following resource in Openshift Web console.

     1. DeploymentConfig.
     2. PersistentVolumeClaim.
     3. Secret.
     4. Service.

- In the Topology view click the MongoDB Instance. Click on the service will get connection IP and Port details.


## Load the data into MongoDB POD

- Execute the following command with sample data.
- Give respective details such `POD NAME, COLLECTION NAME, MongoDB URI`  while executing command.

   ```bash
   oc exec -it ${POD} -- bash -c "mongoimport  -u admin -p password --db ${COLLECTION-NAME} --collection user --jsonArray --mode insert --drop" < sample.json
   #mongoimport --uri ${URI} --collection user --file users.json --jsonArray --mode insert --drop
   ```
